PROJECT OVERVIEW:
-----------------
Lume is a global payroll platform built on the Stellar blockchain network that 
enables businesses to pay international teams with 90% lower fees compared to 
traditional methods like SWIFT and wire transfers. The platform provides instant 
cross-border payments with 5-second transaction finality.

TECH STACK:
-----------
‚Ä¢ Frontend: Next.js 16 (React 19), TypeScript
‚Ä¢ Styling: Tailwind CSS 4
‚Ä¢ Animations: Framer Motion
‚Ä¢ Charts & Visualization: Recharts
‚Ä¢ Blockchain: Stellar SDK (@stellar/stellar-sdk v13.3.0), Stellar Wallets Kit (@creit.tech/stellar-wallets-kit)
‚Ä¢ Wallet Support: Multi-wallet (Freighter Desktop/Mobile, Albedo, xBull, WalletConnect)
‚Ä¢ Database: Supabase (PostgreSQL) with SSR support
‚Ä¢ State Management: Zustand
‚Ä¢ Icons: Lucide React
‚Ä¢ Notifications: Sonner (toast notifications)
‚Ä¢ CSV Export: json-2-csv (for transaction history exports)
‚Ä¢ External APIs: CoinGecko API (for XLM price data)
