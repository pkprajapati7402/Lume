PROJECT OVERVIEW:
-----------------
Lume is a global payroll platform built on the Stellar blockchain network that 
enables businesses to pay international teams with 90% lower fees compared to 
traditional methods like SWIFT and wire transfers. The platform provides instant 
cross-border payments with 5-second transaction finality.

TECH STACK:
-----------
• Frontend: Next.js 16 (React 19), TypeScript
• Styling: Tailwind CSS 4
• Animations: Framer Motion
• Charts & Visualization: Recharts
• Blockchain: Stellar SDK (@stellar/stellar-sdk v13.3.0), Freighter Wallet API (@stellar/freighter-api v6.0.1)
• Database: Supabase (PostgreSQL) with SSR support
• State Management: Zustand
• Icons: Lucide React
• Notifications: Sonner (toast notifications)
• CSV Export: json-2-csv (for transaction history exports)
• External APIs: CoinGecko API (for XLM price data)
