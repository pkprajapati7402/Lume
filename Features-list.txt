PROJECT OVERVIEW:
-----------------
Lume is a global payroll platform built on the Stellar blockchain network that 
enables businesses to pay international teams with 90% lower fees compared to 
traditional methods like SWIFT and wire transfers. The platform provides instant 
cross-border payments with 5-second transaction finality.

TECH STACK:
-----------
• Frontend: Next.js 16 (React 19), TypeScript
• Styling: Tailwind CSS 4
• Animations: Framer Motion
• Blockchain: Stellar SDK, Freighter Wallet API
• Database: Supabase (PostgreSQL)
• State Management: Zustand
• Icons: Lucide React

FUNCTIONAL FEATURES (Currently Working):
-----------------------------------------

1. AUTHENTICATION & WALLET CONNECTION
   • Freighter Wallet integration for connecting Stellar accounts
   • Auto-detection of Freighter wallet installation
   • Public key retrieval and session management
   • Wallet disconnect functionality
   • Guest/authorized state management with Zustand

2. NETWORK MANAGEMENT
   • Toggle between Stellar testnet and mainnet
   • Network-aware API calls to appropriate Horizon servers
   • Network indicator in UI showing current active network

3. ACCOUNT BALANCE DISPLAY
   • Real-time balance fetching from Stellar network
   • Display all assets in connected wallet (XLM and custom assets)
   • Auto-refresh every 30 seconds
   • Manual refresh button
   • Network-specific balance display
   • Handles unfunded account states gracefully

4. LIQUIDITY MONITOR (FX RATES)
   • Live orderbook data from Stellar DEX
   • Real-time FX rates between asset pairs (USDC/NGNT default)
   • Bid/Ask price display
   • Spread calculation and percentage display
   • Liquidity stability indicator
   • Auto-refresh functionality
   • Error handling for non-existent trading pairs

5. SAVINGS CALCULATOR
   • Tracks last 30 days of USDC transactions
   • Calculates savings vs traditional bank wire transfers (5% fee)
   • Animated counter display for savings amount
   • Transaction count tracking
   • Auto-refresh every 60 seconds
   • Handles accounts with no transaction history

6. NETWORK STATISTICS
   • Live Stellar network stats using custom hook
   • Network speed/latency monitoring
   • Base fee display (in stroops and XLM)
   • Auto-refresh every 30 seconds

7. EMPLOYEE DIRECTORY (Full CRUD)
   • View all employees in searchable table format
   • Add new employees with form (name, wallet, role, department, asset)
   • Delete employees with confirmation
   • Search/filter by name, wallet address, or department
   • Real-time data sync with Supabase
   • Displays employee count and details
   • Supabase integration for data persistence

8. BULK EMPLOYEE UPLOAD
   • CSV file drag-and-drop upload interface
   • CSV parsing and validation
   • Preview parsed data before submission
   • Batch insert to Supabase database
   • Template CSV download functionality
   • Error handling for invalid CSV formats
   • Support for columns: Full_Name, Wallet_Address, Role, Department, Preferred_Asset

9. LANDING PAGE
   • Hero section with animated gradient text
   • Feature showcase with icons and descriptions
   • Pricing information display
   • Call-to-action buttons
   • Responsive navigation bar
   • Footer with links and information
   • Smooth scroll animations with Framer Motion
   • Wallet connection from landing page

10. RESPONSIVE DASHBOARD LAYOUT
    • Mobile-responsive sidebar navigation
    • Collapsible mobile menu
    • Section-based navigation (Overview, Pay Employee, Bulk Upload, Directory)
    • Header with network toggle and wallet info
    • Logout functionality
    • Smooth transitions between sections

11. CUSTOM 404 PAGE
    • Branded error page with gradient styling
    • "Return to Homepage" navigation button
    • Consistent design with application theme

12. DATABASE SCHEMA
    • Complete PostgreSQL schema for Supabase
    • Tables: employees, payouts, settings
    • UUID primary keys and proper indexes
    • Timestamp tracking (created_at, updated_at)
    • Proper foreign key relationships
